# Walmart_Sales-Project# Walmart Sales Data Analysis Project

## Overview
This project performs end-to-end sales analysis on Walmart data using SQL and Python.

## Tools & Technologies
- SQL (SQLite)
- Python (Pandas)
- Google Colab
- GitHub

## Dataset
- Walmart sales dataset (cleaned)
- ~9,969 rows and 11 columns

## Key Business Questions Answered
- Difference between payment methods based on number of transactions and quantity sold
- Highest rated product category in each branch
- Busiest day for each branch based on transactions
- Sales and customer behavior analysis

## Approach
1. Data cleaning using Python
2. Data stored in SQL database
3. SQL queries executed using Python
4. Insights extracted for business decisions

## Outcome
This project demonstrates strong fundamentals in SQL querying, data analysis, and problem-solving.
